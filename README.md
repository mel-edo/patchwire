# Patchwork

A PipeWire audio output router for Linux. Automatically links audio sinks to your default output so you can share playback across multiple devices simultaneously ex. wireless earbuds + wired earbuds, speakers + headphones, etc.

Instead of manually wiring ports in Helvum every time you connect a device, patchwork watches the PipeWire graph and handles it for you, with per-sink toggles, named profiles, and a GTK4 GUI.
